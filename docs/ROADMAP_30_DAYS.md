# 30-Day Roadmap

## Week 1
- Finalize requirements and user roles
- Implement DB schema and migrations
- Setup backend project structure and auth
- Build machine registry APIs

## Week 2
- Build maintenance plan APIs
- Implement auto work order generation scheduler
- Build work order lifecycle endpoints
- Seed demo data for 10 machines

## Week 3
- Implement failure log APIs
- Build dashboard summary + MTBF/MTTR calculations
- Add CSV exports
- Add basic alert hooks (email/webhook)

## Week 4
- Frontend pages: machines, plans, work orders, dashboard
- Role-based permission checks in UI and API
- Testing pass (unit + integration)
- Dockerized deployment and MVP release v0.1.0

## Deliverables by Day 30
- Working MVP with preventive scheduling
- Dashboard with actionable reliability metrics
- Deployment guide and sample dataset
- Public open-source repo ready for contributors
